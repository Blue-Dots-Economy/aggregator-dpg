# Draft issues for the Aggregator DPG MVP

These markdown files are the source of truth for the GitHub issues we will create. Each file corresponds to one issue. File naming:

- Epics: `<area>/P-NN-<slug>.md` (platform), `product/PH-N-<slug>.md` (phase epic), `post-mvp/X-NN-<slug>.md`
- Features: `<area>/F-NN.M-<slug>.md`
- Tasks: `<area>/T-NN.M.K-<slug>.md`

Numbering matches the spec at `docs/superpowers/specs/2026-04-21-aggregator-issues-design.md` so cross-reference is trivial.

## Directory layout

- `platform/` — Platform epics P-01 … P-20 with their features and tasks
- `product/` — Product (phase) epics Φ1 … Φ4 with their features and tasks
- `post-mvp/` — Post-MVP tracking epic X-01 with its stub issues
